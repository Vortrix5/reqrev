# Icon Placeholders

This directory should contain the extension icons:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Quick Fix

If you don't have icons yet, you can:

1. **Create simple placeholder icons** using any image editor
2. **Use an online icon generator** like https://www.favicon-generator.org/
3. **Temporarily comment out the icons** in `manifest.json`

## Design Guidelines

For ReqRev branding:
- Use the accent blue color: `#7aa2ff`
- Include a checklist or requirement symbol
- Maintain GitHub's visual style

The extension will work without icons, but Chrome will show a default puzzle piece icon.
